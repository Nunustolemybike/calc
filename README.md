# CALC

Use the library calc to make ur calcul easier !

## Prerequisites 

Download the files and import it.

## Exemple

Exemple of addition :

var calc = require('./calc.js');
calc.add(1,2) #result=3

Exemple of substract :

var calc = require('./calc.js');
calc.substract(2,1) #result=1

Exemple of multiply :

var calc = require('./calc.js');
calc.multiply(1,2) #result=2
